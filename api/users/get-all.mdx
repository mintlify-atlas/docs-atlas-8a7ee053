---
title: "Get All Users"
api: "GET /api/users"
description: "Retrieve a list of all users from the database"
---

This endpoint retrieves all users registered in the Pet Adoption system.

## Response

<ResponseField name="status" type="string">
  Status of the response (e.g., "success")
</ResponseField>

<ResponseField name="payload" type="array">
  Array of user objects
  
  <ResponseField name="_id" type="string">
    Auto-generated MongoDB ObjectId
  </ResponseField>
  
  <ResponseField name="first_name" type="string">
    User's first name
  </ResponseField>
  
  <ResponseField name="last_name" type="string">
    User's last name
  </ResponseField>
  
  <ResponseField name="email" type="string">
    User's email address (unique)
  </ResponseField>
  
  <ResponseField name="password" type="string">
    User's hashed password
  </ResponseField>
  
  <ResponseField name="role" type="string" default="user">
    User's role (e.g., "user" or "admin")
  </ResponseField>
  
  <ResponseField name="pets" type="array">
    Array of pet references associated with the user
    
    <ResponseField name="_id" type="string">
      MongoDB ObjectId reference to a Pet
    </ResponseField>
  </ResponseField>
</ResponseField>

## Example Request

```bash curl
curl --request GET \
  --url http://localhost:8000/api/users
```
