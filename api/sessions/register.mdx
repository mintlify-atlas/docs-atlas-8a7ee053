---
title: 'Register User'
api: 'POST /api/sessions/register'
description: 'Create a new user account with email and password'
---

## Request Body

<ParamField body="first_name" type="string" required>
  User's first name
</ParamField>

<ParamField body="last_name" type="string" required>
  User's last name
</ParamField>

<ParamField body="email" type="string" required>
  User's email address (must be unique)
</ParamField>

<ParamField body="password" type="string" required>
  User's password (will be hashed before storage)
</ParamField>

## Response

<ResponseField name="status" type="string">
  Status of the request - "success" or "error"
</ResponseField>

<ResponseField name="payload" type="string">
  The newly created user's ID (returned on success)
</ResponseField>

<ResponseField name="error" type="string">
  Error message (returned on failure)
</ResponseField>

## Example Request

```bash cURL
curl -X POST https://api.petadoption.com/api/sessions/register \
  -H "Content-Type: application/json" \
  -d '{
    "first_name": "John",
    "last_name": "Doe",
    "email": "john.doe@example.com",
    "password": "securePassword123"
  }'
```

## Example Response

```json Success (200)
{
  "status": "success",
  "payload": "60d5ec49f1b2c72b8c8e4f3a"
}
```

```json Error - Missing Fields (400)
{
  "status": "error",
  "error": "Incomplete values"
}
```

```json Error - User Exists (400)
{
  "status": "error",
  "error": "User already exists"
}
```
