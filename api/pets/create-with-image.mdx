---
title: 'Create Pet with Image'
api: 'POST /api/pets/withimage'
---

Create a new pet in the adoption system with an image upload.

## Request Body

This endpoint accepts `multipart/form-data` with the following fields:

<ParamField body="name" type="string" required>
  Name of the pet
</ParamField>

<ParamField body="specie" type="string" required>
  Species of the pet (e.g., dog, cat, bird)
</ParamField>

<ParamField body="birthDate" type="string" required>
  Birth date of the pet in ISO 8601 format (e.g., "2020-05-15")
</ParamField>

<ParamField body="image" type="file" required>
  Image file of the pet (uploaded as multipart form data)
</ParamField>

## Response

<ResponseField name="status" type="string">
  Response status indicator ("success" or "error")
</ResponseField>

<ResponseField name="payload" type="object">
  The created pet object
  
  <ResponseField name="_id" type="string">
    Unique identifier for the newly created pet
  </ResponseField>
  
  <ResponseField name="name" type="string">
    Name of the pet
  </ResponseField>
  
  <ResponseField name="specie" type="string">
    Species of the pet
  </ResponseField>
  
  <ResponseField name="birthDate" type="string">
    Birth date of the pet in ISO 8601 format
  </ResponseField>
  
  <ResponseField name="adopted" type="boolean">
    Whether the pet has been adopted (default: false)
  </ResponseField>
  
  <ResponseField name="image" type="string">
    Path to the uploaded pet image file
  </ResponseField>
</ResponseField>

<ResponseField name="error" type="string">
  Error message if status is "error" (e.g., "Incomplete values")
</ResponseField>

## Example Request

```bash cURL
curl -X POST http://localhost:8080/api/pets/withimage \
  -H "Content-Type: multipart/form-data" \
  -F "name=Buddy" \
  -F "specie=dog" \
  -F "birthDate=2020-05-15" \
  -F "image=@/path/to/buddy.jpg"
```

## Example Response

### Success Response

```json
{
  "status": "success",
  "payload": {
    "_id": "507f1f77bcf86cd799439011",
    "name": "Buddy",
    "specie": "dog",
    "birthDate": "2020-05-15T00:00:00.000Z",
    "adopted": false,
    "image": "/public/img/buddy-1234567890.jpg"
  }
}
```

### Error Response

```json
{
  "status": "error",
  "error": "Incomplete values"
}
```
