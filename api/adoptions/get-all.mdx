---
title: "Get All Adoptions"
api: "GET /api/adoptions"
description: "Retrieve a list of all pet adoptions"
---

This endpoint returns all adoption records in the system, showing which users have adopted which pets.

## Request

```bash
curl -X GET https://your-api-domain.com/api/adoptions
```

## Response

<ResponseField name="status" type="string" required>
  The status of the response (e.g., "success")
</ResponseField>

<ResponseField name="payload" type="array" required>
  Array of adoption records
  
  <ResponseField name="_id" type="string">
    Unique identifier for the adoption record
  </ResponseField>
  
  <ResponseField name="owner" type="string">
    Reference to the User ID who adopted the pet
  </ResponseField>
  
  <ResponseField name="pet" type="string">
    Reference to the Pet ID that was adopted
  </ResponseField>
</ResponseField>
